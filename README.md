# 🔒 Cyber Career Roadmap

A gamified cybersecurity career tracker built as a Progressive Web App. Track every cert, tool, and skill on the path from zero to DoD Security Engineer.

LIVE APP 🔥 → https://thestaticcore.github.io/The-Cyber-Checkmark/

---

## ✨ Features

- 50 items across 4 phases
- XP system with 10 levels — Iron → Phantom
- Difficulty ratings per item (1-3)
- Estimated study hours per item and phase
- Progress rings per phase and overall
- Dark and light mode
- PWA — installs on phone and desktop
- Full offline support
- Saves progress automatically
